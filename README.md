## 📌 Project Overview

A full-stack food delivery web application inspired by Zomato. Users can browse food items, add them to cart, and place orders. Includes authentication, database integration, and responsive UI.

---

## 🧠 Tech Stack

* Backend: Flask (Python)
* Database: SQLite / PostgreSQL
* Frontend: HTML, CSS, Bootstrap
* Deployment: Render

---

## 🔥 Features

* User Registration & Login (Secure Authentication)
* Browse Food Menu
* Add to Cart
* Place Orders
* Session-based Cart Management
* Clean UI with Bootstrap

## ⚙️ Installation & Setup

```bash
git clone https://github.com/your-username/campus-food-delivery-system.git
cd campus-food-delivery-system
pip install -r requirements.txt
python app.py
```

## 🌍 Deployment (Render)

* Build Command: pip install -r requirements.txt
* Start Command: gunicorn app:app
